# 🚀 Sistema Antifraude Pix (Backend Java)

Este projeto simula um sistema backend de validação de transações Pix, com foco em regras de risco e análise antifraude.

## 📌 Objetivo

Desenvolver uma aplicação backend capaz de processar e validar transações financeiras, aplicando regras de negócio para identificar possíveis fraudes.

## 🧠 Como funciona

O sistema recebe uma transação Pix e aplica regras de validação, como:

* Verificação de valores suspeitos
* Análise de comportamento da transação
* Definição de status (APROVADO / RECUSADO)

## ⚙️ Tecnologias utilizadas

* Java
* Programação Orientada a Objetos
* Estrutura em camadas (Controller, Service)

## 🔍 Conceitos aplicados

* Backend
* APIs REST (simulação)
* Regras de negócio
* Lógica antifraude

## 💡 Diferencial

Este projeto simula, em nível simplificado, como funcionam sistemas antifraude utilizados por instituições financeiras, que analisam padrões e comportamentos para reduzir riscos em transações Pix.

## 🤖 Uso de IA

Durante o desenvolvimento, utilizei Inteligência Artificial como apoio para estruturar ideias, validar abordagens e acelerar o aprendizado, mantendo total entendimento da lógica implementada.
